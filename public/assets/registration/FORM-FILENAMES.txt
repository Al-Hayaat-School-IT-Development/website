PDFs in this folder must match `requirements.forms[].file` in `src/content/admissions.json`.

Current files (spaces are OK — the app builds URLs with encodeURIComponent):

  Registration Form.pdf
  PAD Agreement.pdf
  Financial Agreement.pdf
  Parental Agreement.pdf
  Liability Waiver.pdf

Production: point NEXT_PUBLIC_REGISTRATION_FORMS_BASE_URL at the blob path prefix (no trailing slash), e.g.

  NEXT_PUBLIC_REGISTRATION_FORMS_BASE_URL=https://alhayaatassetsprod.blob.core.windows.net/school-registration

When Azure CDN (or Front Door) is enabled, switch that value to the CDN endpoint + same path segment; filenames in admissions.json stay the same.
